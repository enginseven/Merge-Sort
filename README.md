# Merge-Sort

Merge Sort :  Bir listeyi her adımda parçaya ayırıp tek eleman kalıncaya kadar bölüyor. Böldükten sonra sıralı bir şekilde bize sunar.
[16, 21, 11, 8, 12, 22] 0 adım
[16, 21, 11]			    [8, 12, 22]

[16, 21]     [11]			[8, 12]        [22]

[16]  [21]   [11]			[8]  [12]      [22]

[16, 21]     [11]			[8, 12]      [22]

[16,  11, 21]			[8, 12, 22]    

    [8, 11, 12, 16, 21, 22 ]

O(nlogn) dir.
